A STTweet is an Object that represents a single Tweet.

Instance Variables
	createdAt:		<String>
	entities:		<Object>
	id:		<Integer>
	text:		<String>
	user:		<STUser>

createdAt
	- time when it was postetd

entities
	- Metadata and Additional Information

id
	- id given by Twitter

text
	- content of the tweet

user
	- used that created the tweet
