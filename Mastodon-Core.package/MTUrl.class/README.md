A STUrl is an Enities that contains diffrent versions of urls mentioned in a tweet.

Instance Variables
	displayUrl:		<String>
	expandedUrl:		<String>
	shortUrl:		<String>

displayUrl
	- Url as it should be shown to user

expandedUrl
	- display url with prefix

shortUrl
	- compressed url as it is in the uriginial text of the tweet
